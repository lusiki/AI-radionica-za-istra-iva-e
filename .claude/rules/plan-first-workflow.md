# Plan-First Workflow  (always-on)

For any non-trivial task — roughly: it touches **more than 3 files**, takes **more than ~30 minutes**,
or is a **multi-step workflow** — do NOT start editing.

1. **Enter plan mode** (read-only). Explore, then draft a short plan.
2. **Save the plan** to `quality_reports/plans/YYYY-MM-DD_<slug>.md` so it survives context compaction.
3. **Show the plan and get approval** before touching a single file.
4. On approval, switch to **contractor mode**: execute end-to-end, verify, review, and come back only
   at a real decision point or when the job passes inspection.

Trivial edits (a typo, one bullet) skip this — just do them.
