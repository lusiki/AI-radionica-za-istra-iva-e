# Quality Gates  (loads on slides & resources)

Score every reviewed file 0–100 and read it like a traffic light:

| Score | Meaning |
|------|---------|
| **95+** | Excellence (aspirational) |
| **90+** | Ready to ship / present |
| **80+** | Safe to commit / save progress |
| **< 80** | **Blocked** — fix the criticals first |

Sample deductions (adapt as needed):

- Content slide overflows / runs off the slide — **−15**
- Unverifiable or wrong claim, number, or citation — **−15**
- Off-message framing (sells "productivity"/hype instead of delegation) — **−10**
- Same concept named two different ways across slides — **−5**
- Typo or grammar error — **−5**
- A slide with no clear single takeaway — **−5**

Report the score with the findings. Below 80, do not offer to commit.
